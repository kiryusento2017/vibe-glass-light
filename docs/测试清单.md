# Claude Code Light 测试清单

交代测试者：下载 `claude-traffic-light.exe`，本机需装 Claude Code。

## 必测

- [ ] 双击启动，挂件出现在屏幕顶部居中
- [ ] 透过胶囊能看到桌面被折射扭曲（不是黑框）
- [ ] 托盘有红绿灯图标，exe 文件本身也有图标
- [ ] 打开 Claude Code 对话 → 绿/黄/红灯正常切换
- [ ] 关掉 Claude Code → 3 秒内灭灯
- [ ] 按住胶囊可拖动，松手位置不变
- [ ] 点胶囊外的透明区域拖不动
- [ ] 按住拖动有果冻形变，松手 Q 弹回稳
- [ ] 右键菜单：调整大小 / 开机自启 / 隐藏 / 固定位置 / 重置 / 退出
- [ ] 调整大小滑块窗：100%~2000% 无极缩放、不卡、清晰、顶边不动
- [ ] 放大后退出重开，大小和位置不变
- [ ] 重置大小和位置 → 回默认 100% 居中
- [ ] 开机自启：勾选→注册表有记录→取消勾选→记录消失
- [ ] 隐藏窗口→托盘还在→右键托盘显示回来
- [ ] 退出→挂件和托盘一起消失

## 选测

- [ ] 再双击 exe → 不会启动第二个
- [ ] 拖到屏幕边缘 / 副屏 → 不崩
- [ ] HiDPI（125%/150%）→ 不糊
- [ ] 编辑 `glass-tuning.json` 改参数 → 0.5s 内视觉变化
- [ ] 长时间挂着不管 → 不崩不卡
